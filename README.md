# dig_dug
Fundamentals 2 Final Project 


Special Libraries:

The SDL image, and SDL ttf libraries are required. 


How to install:

Just use gnu Make from the main folder. 


Controls:

The character is moved via the arrow keys on the keyboard.  


Known bugs:

When walking along the right edge, the sprite may clip outside of the frame.
Sometimes on spawn, the monsters may spawn on top of you so that you do not have a chance to esacape.  
The same may happen with the flower, causing you to win immediatley.
